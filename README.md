# ATS NX Custom Launcher

Lanzador modular en PowerShell para entornos Siemens NX.

## Características
- Arquitectura modular (Core aislado del usuario).
- Detección dinámica de ejecutables de NX desde el Registro de Windows.
- Mapeo dinámico y condicional de variables de entorno (CAD/CAM, Postprocesadores, Plantillas).
- Asistente de configuración de Red (Configura 1 vez, aplica a todos los equipos).

## Uso
Ejecutar el archivo `Run-ATSLauncher.bat`.
